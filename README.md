# WaveTable Generator [Final Master]

2048サンプル×256フレームの業界標準規格と8倍オーバーサンプリングを採用した、プロ仕様に近いウェーブテーブル生成ツールです。[file:1]  
Vital などのウェーブテーブル対応シンセサイザー向けに設計されています。

## 特徴

- 全54種類の波形ライブラリを搭載。
- 全23種類のモジュレーション／特殊機能を実装。
- 数学的アルゴリズムによるリアルタイム合成。録音ベースではない高品位な波形生成。[file:1]
